A modern e-commerce style landing page for a gaming hardware store.

## Description
This project is a static frontend interface that presents a complete store experience for gaming products. It is designed to showcase clean layout structure, product-focused sections, and interactive visual effects using pure HTML and CSS.

## Features
- Hero section with strong call-to-action buttons
- Multi-level navigation with dropdown menu
- Product sections: Trending Tech and Recommended For You
- Customer testimonial area and Tech News cards
- Service highlights (shipping, financing, support)
- Newsletter subscription block and full footer links
- Hover animations and transition effects across key UI components

## Tech Stack
- HTML5
- CSS3
- Font Awesome (local files in `css/all.css` and `webfonts/`)
- Google Fonts (`Clicker Script`)

## Getting Started
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Open the project folder.
3. Run `index.html` directly in your browser, or use VS Code Live Server for better development workflow.
